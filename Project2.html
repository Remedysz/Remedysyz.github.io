Software Bug Reporting Web Application

  • Software Debug Project

    This project is demonstrable through the the link above.

    This project was mainly to test my ability to do a simple web application using different technologies. The technologies included in this project are as followed : Auth0, JavaScript, ASP.net Core, SQLite, jQuery, Bootstrap, MVC architectural design. The application is a simulated software bug logging web app that allows users with validated accounts to create, edit, and submit bugs. The data they submit or edit about the bug are saved locally on the web app through SQLite. The database is easily scalible by replacing it with a SQL server hosted database. The website is fully functional and scaled visually to most screen resolutions by using bootstrap.
